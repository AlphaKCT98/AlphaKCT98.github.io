References and further learning

References and Learning Resources

This page collects the sources from the original paper and adds more official documentation so the site can support stronger research and deeper study.

Official documentation and standards

These are the strongest sources for technical study.

  1. MDN Web Docs: Overview of HTTP - clear overview of HTTP as a client-server protocol and how a document is built from many resources.
  2. MDN Web Docs: HTTP Messages - explains requests, responses, headers, and bodies.
  3. MDN Web Docs: HTTP Response Status Codes - quick reference for common HTTP results and their categories.
  4. MDN Web Docs: Evolution of HTTP - useful summary of major protocol changes.
  5. IETF RFC 9114: HTTP/3 - formal standard describing HTTP/3 over QUIC.
  6. ICANN: The Domain Name System - official explanation of how DNS helps users navigate the internet.
  7. Apache HTTP Server Documentation - official documentation for Apache.
  8. NGINX Documentation - official docs for request handling, proxying, HTTPS, and load balancing.
  9. Microsoft IIS Documentation - official overview and administration resources for IIS.

Supporting background sources

These are useful for history and broader context.

  1. Apache HTTP Server - historical background on Apache's role in the web.
  2. Common Gateway Interface - background on an early approach to dynamic content.
  3. HTTP - broad background on the protocol.
  4. NCSA HTTPd - useful history for early server development.
  5. TechTarget: What Is a Web Server? - beginner-friendly explanation of web server purpose and roles.
  6. Web Server - broad background on web server hardware and software.

References and Learning Resources

This page collects the sources used in the original research paper and adds a short explanation of why each one is useful.

References used for the project

  1. Apache HTTP Server , background on Apache and its role in web history.
  2. Common Gateway Interface , explains one of the earliest ways servers generated dynamic content.
  3. MDN Web Docs: Evolution of HTTP , a strong overview of HTTP version changes.
  4. HTTP , general history and background of the protocol.
  5. NCSA HTTPd , useful for understanding early server development.
  6. TechTarget: What Is a Web Server? , clear explanation of the purpose of web servers.
  7. What Is a Web Server and How It Works , adds a beginner friendly discussion of function and security.
  8. Web Server , broad background on web server hardware and software.
  9. HTTP 1 vs HTTP 2 vs HTTP 3 , compares major versions in simple language.

Best starting points for new learners

  • MDN: best for technical explanations that are still readable.
  • TechTarget: useful for quick definitions and beginner summaries.
  • Wikipedia: useful for history and cross-referencing related topics.

Research note

This site is based on the first essay version of the project, so the content is intentionally focused on core ideas. Later milestones can expand the research with more detail, more examples, and stronger visuals.

Project goals

What this site is trying to do well.

  • Turn research writing into a multi-page website that is easy to explore.
  • Practice semantic HTML, layout, navigation, responsive styling, and light JavaScript.
  • Create a site that feels more like a real study resource than a plain assignment upload.
Skills practiced in this project
Skill Area How it appears in this site
Semantic HTML Pages use headers, navigation, sections, figures, lists, and tables.
CSS Styling One shared stylesheet controls layout, spacing, hover effects, colors, and responsiveness.
Content Organization The research topic is split into focused pages so readers can move through the material more easily.
User Experience The site uses a consistent menu, readable spacing, and responsive layouts for smaller screens.

Author statement

I chose this topic because people use the internet all the time without usually thinking about the systems that make it work. Building this site helped me take a topic that could have felt abstract and break it into smaller pages that are easier to read, understand, and study from later.

This page introduces the author, the purpose of the project, and the skills being practiced through this website.

Mini Resume

Name: Keyvon Thompson

Course: Intro to Web Development

Project Topic: Web servers and HTTP

Current Focus: Learning how websites work behind the scenes and how to present research through HTML and CSS.

Project Goals

  • Turn research writing into a clear multi page website.
  • Practice semantic HTML, navigation, layout, and CSS styling.
  • Create a site that is simple, readable, and easy to expand in later milestones.
Skills practiced in this project
Skill Area How it appears in this site
Semantic HTML Pages use headers, navigation, sections, figures, lists, and tables.
CSS Styling One shared stylesheet controls layout, spacing, hover effects, colors, and responsiveness.
Content Organization The research topic is split into focused pages so readers can move through the material more easily.
User Experience The site uses a consistent menu, readable spacing, and responsive layouts for smaller screens.

Author statement

I chose this topic because most people use the internet constantly but do not usually stop to think about the systems that make it work. Building this website helped me explain those systems in a way that feels more visual and organized than a traditional essay.
